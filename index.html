Welcome to NCAA Basketball Tournament Prediction Analysis

This is a page provided by GitHub to present our project related with analysis of the games played during the famous NCAA College Basketball Tournament played each year in March in USA. This is a MSc course group project from the Data Analytics module at Queen Mary University.

The team

We love machine learning, big data and sports. When you combine them what do you get? - Sports Analytics. We build this team not only with the idea to get a good note for the degree, but to learn, improve and challenge ourselves. Here we are:
